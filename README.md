# Praktikum 1-4 Pemrograman Web 2 - CodeIgniter 4

Website artikel sederhana dengan fitur CRUD dan authentication menggunakan **CodeIgniter 4**

<img width="1366" height="768" alt="Screenshot at 2026-04-27 21-27-36" src="https://github.com/user-attachments/assets/31cfebb0-3e4a-4dce-924c-a6944a95e80a" />


## ✨ Fitur

- Halaman publik: Home (daftar artikel), About, Service, FAQ  
- Admin login (autentikasi session)  
- CRUD artikel (Create, Read, Update, Delete)  
- Dashboard admin dengan tabel artikel  
- Tampilan modern & timeless (Tailwind CSS)  
- Responsif
